CHAPITRE2 : LES ATTRIBUTS

On a trois types d'attributs:

  • les attributs spécifiques
  • les attributs globaux
  • les attributs d'évenements

  1. les attributs spécifiques:
    1. Définitions :
      • Tous les éléments HTML peuvent avoir des attributs
      • Les attributs fournissent des informations supplémentaires sur les éléments
      • Les attributs sont toujours spécifiés dans la balise de début
      • Les attributs se présentent généralement sous forme de paires nom / valeur comme: name = "value"
    2. Exemples:
      • L'attribut href spécifier l'URL de la page
      • L'attribut src de img spécifie le chemin vers l'image à afficher
      • Les attributs width et height de img fournissent des informations sur la taille des images
      • L'attribut alt de img fournit un texte alternatif pour une image
  2. les attributs globaux:
    1. Définitions :

      2. Les attributs globaux sont des attributs qui peuvent être utilisés avec tous les éléments HTML.

    2. Résumé
      • L'attribut accesskey Spécifie une touche de raccourci pour activer ou focaliser un élément
      • L'attribut class Spécifie un ou plusieurs noms de classe pour un élément (fait référence à une classe dans une feuille de style)
      • L'attribut contenteditable Spécifie si le contenu d'un élément est modifiable ou non
      • L'attribut data-* Utilisé pour stocker des données personnalisées privées sur la page ou l'application
      • L'attribut dir Spécifie la direction du texte pour le contenu d'un élément
      • L'attribut draggable Spécifie si un élément peut être déplacé ou non
      • L'attribut hidden Spécifie qu'un élément n'est pas encore ou n'est plus pertinent
      • L'attribut id Spécifie un identifiant unique pour un élément
      • L'attribut lang Spécifie la langue du contenu de l'élément
      • L'attribut spellcheck Spécifie si l'orthographe et la grammaire de l'élément doivent être vérifiées ou non
      • L'attribut style Spécifie un style CSS inline pour un élément
      • L'attribut tabindex Spécifie l'ordre de tabulation d'un élément
      • L'attribut titleSpécifie des informations supplémentaires sur un élément
      • L'attribut translateSpécifie si le contenu d'un élément doit être traduit ou non
